Kaepi Web Solutions - Vite + React Single Page
----------------------------------------------------

How to use:
1) Unzip the project.
2) In project root run:
   npm install
3) Start dev server:
   npm run dev
4) Open http://localhost:5173 (or the URL Vite shows)

Files created:
 - index.html
 - package.json
 - src/main.jsx
 - src/App.jsx
 - src/App.module.css
 - public/images/* (placeholder images)
 - README.md

Where to paste your images:
 - Place hero images at public/images/hero1.jpg ... hero4.jpg
 - Replace /images/about.jpg, /images/og-image.jpg, /images/work1.jpg ... work4.jpg
 - Favicon at public/favicon.ico

Notes:
 - This is a lightweight starter. Replace placeholder images with your real images.
 - To deploy, build with `npm run build` and upload the `dist/` folder to your hosting.
 - For WordPress sites, you can use the same content and imagery when creating WP pages.
